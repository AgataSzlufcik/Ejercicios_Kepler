# Ejercicios_Kepler
Limpieza y preparación de datos. Tratamiento de valores nulos, duplicados, formatos incorrectos y errores comunes, con enfoque práctico en calidad del dato como base del análisis.
Teoria y practica.

Ejercicios practicos de entrega estan ubicados en la siguiente orden.
1. e1_analisis_basico_de_datos.py 
2. e2_clasificador_de_prestamos.py
3. e3_analisis_de_categorias.py
4. e4_procesador_de_datos.py
5. e5_mini_proyecto_integrador.py
